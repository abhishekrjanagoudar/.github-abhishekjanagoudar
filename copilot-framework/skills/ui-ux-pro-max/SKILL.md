# UI/UX Pro Max (Copilot Framework)

## Design database
- 67 UI styles (Glass, Brutalist, Neu...)
- 161 color palettes by industry
- 57 typography pairings (Google Fonts)
- 99 UX guidelines (accessibility, flow)
- 25 chart types for data visualization
- 161 reasoning rules for style matching

## Design system generation (v2.0)
1. Multi-domain search across 5 parallel domains: products, styles, colors, patterns, typography
2. Reasoning engine matches product to UI category and applies style priorities
3. Filters anti-patterns automatically
4. Outputs: pattern, style, palette, typography, effects, and checklists

## Supported stacks
Web: React, Next.js, Vue, Svelte, Astro
Mobile: Swift UI, React Native, Flutter
Libraries: shadcn/ui, Nuxt UI, Tailwind

## How it works
1. Describe your product + audience
2. Skill selects style + palette + fonts
3. Generates full design system tokens
4. Applies to every component it builds

## Pre-delivery checklist
- Contrast ratio >= 4.5:1 (WCAG AA)
- Touch targets >= 44px minimum
